# Project Overview

## Project Title

Sequence Detector (1011) using Verilog HDL

---

## Objective

The objective of this project is to design and simulate Finite State Machine (FSM) based sequence detectors capable of detecting the binary sequence **1011**. The project demonstrates the implementation of two different FSM models:

- Mealy Sequence Detector (Overlapping)
- Moore Sequence Detector (Non-overlapping)

The designs are implemented in Verilog HDL and verified through behavioral simulation using AMD Vivado.

---

## Problem Statement

Design a digital circuit that continuously monitors a serial input stream and generates an output whenever the sequence **1011** is detected.

The detector should correctly identify the sequence while following the required FSM behavior.
