## Simulation Verification

The simulation results confirm that both sequence detectors operate according to their respective finite state machine (FSM) designs.

### Mealy Sequence Detector

- The output is generated immediately when the final bit of the sequence **1011** is received.
- Since the output depends on both the current state and the input, the detection pulse occurs during the same clock cycle as the last input bit.
- The design correctly detects overlapping occurrences of the sequence.

### Moore Sequence Detector

- The output becomes HIGH only after the FSM enters the detection state (**S4**).
- Since the output depends only on the current state, the detection pulse appears one clock cycle later than in the Mealy implementation.
- The design correctly follows Moore FSM behavior.
